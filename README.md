# 👑 폴리 컴패스 (Poli Compass)

> **"정당은 자신의 신념을 지켜야 할까, 아니면 선거 승리를 위해 중도층에 가까워져야 할까?"**
> 본 웹 애플리케이션은 중학교 3학년 사회(정치와 사회) 및 고등학교 사회·문화/정치와 법 수업에서 활용할 수 있는 **교사용 정치·선거 시뮬레이터**입니다.

이 시뮬레이터는 학생들이 직접 정당을 창당하고, 가상의 국가인 **'세종국'**에 분포한 100명의 유권자를 확보하기 위해 정책 예산을 배분하며 경쟁하는 수업용 실험 웹앱입니다.

---

## 1. 수업 목표
- **이념 스펙트럼의 이해**: 경제적(복지 vs 자유) 및 사회적(자유 vs 질서) 축을 기반으로 한 2차원 정치적 성향 분포를 이해합니다.
- **다운스의 중위투표자 정리 체험**: 다수대표제 하에서 왜 정당들의 정책이 중도층으로 수렴하게 되는지 직관적으로 파악합니다.
- **선거 전략의 다변화 분석**: 확고한 지지층 결집(이념 선명성)과 중도층 확장(외연 확장) 사이의 정책 딜레마를 분석합니다.
- **정치공학의 한계 이해**: 후보 단일화나 합당이 지지율의 단순 합산으로 이어지지 않으며, 오히려 이념 희석으로 인해 유권자들의 이탈이 일어날 수 있음을 깨닫습니다.

---

## 2. 화면 구성 및 사용 환경
- **교사용 전자칠판 최적화**: 대형 스크린에서도 유권자 분포와 득표 현황이 선명하게 보이도록 고대비 다크 테마(미드나잇 블루)와 네온 하이라이팅을 사용했습니다.
- **단계별 가이드라인**: 상단 타임라인을 통해 현재 시뮬레이션 진척을 확인할 수 있으며, 하단 컨트롤바에서 각 단계에 필요한 입력 폼을 제공합니다.
- **교사용 힌트 패널**: 각 수업 단계에서 교사가 학생들에게 던질 수 있는 추천 발문이 접이식 패널로 제공됩니다.
- **전체화면 발표 모드**: 오른쪽 상단의 확장 버튼을 눌러 브라우저 주소창을 숨기고 전자칠판 발표에 집중할 수 있습니다.

## 3. 핵심 시뮬레이션 규칙 및 계산 방식

### ① 정당 이념 및 예산 좌표 연산
각 정당의 최종 이념 좌표 $(X_{final}, Y_{final})$는 전통적 정체성을 나타내는 **초기 이념 좌표(40%)**와 100억의 **정책 예산 좌표(60%)**를 혼합하여 결정됩니다.
(반영 비율은 교사용 세부 설정에서 조정 가능합니다.)

#### [5대 정책 분야 벡터 가중치]
- **청년·일자리**: 경제 진보, 사회 자유 $(x: -50, y: +30)$
- **노인·복지**: 경제 진보, 사회 질서 $(x: -70, y: -30)$
- **부동산·세금**: 경제 보수, 사회 질서 $(x: +60, y: -10)$
- **환경·기후**: 경제 진보, 사회 자유 $(x: -40, y: +70)$
- **국방·치안**: 경제 보수, 사회 질서 $(x: +10, y: -70)$

$$\text{예산 좌표 } (X_{budget}, Y_{budget}) = \sum \left( \frac{\text{분야별 배분 예산}}{\text{총예산 } 100\text{억}} \times \text{분야별 벡터} \right)$$
$$\text{최종 좌표 } (X_{final}, Y_{final}) = (X_{initial} \times 0.4) + (X_{budget} \times 0.6)$$

### ② 유권자 지지 매칭 알고리즘
- **유클리드 거리**: 유권자의 개인 좌표와 각 정당의 최종 좌표 간 유클리드 거리를 측정하여 가장 가까운 정당을 지지합니다.
- **선명성 보너스 (Identity Bonus)**: 정당의 최종 좌표가 원점(중도)에서 멀수록(선명하고 극단적일수록), 해당 이념 방향과 유사한 성향을 지닌 특정 외곽 유권자들에게 거리 차감 혜택(최대 22거리)을 제공하여 충성파 지지 효과를 구현합니다.
- **무당층(지지 포기) 필터**: 모든 정당과의 최단 유클리드 거리가 교사가 설정한 한계치(기본값 80)를 넘는 경우, 유권자는 투표를 포기하고 회색 점(무당층)으로 남습니다.

### ③ 단일화/합당 규칙
- 두 정당을 선택하여 합당을 실행하면 신당의 초기 이념과 배분된 정책 예산은 **두 정당의 산술 평균**으로 조정됩니다.
- 조정된 새로운 이념 위치를 기준으로 모든 유권자들이 지지 여부를 재산출하므로, **"두 당의 지지율이 단순 합산되지 않는다"**는 정치 현실을 학생들이 깨닫게 만듭니다.

## 4. 시뮬레이션 단계별 수업 진행 가이드

### [1차시] 정당의 탄생과 이념의 좌표
- 학생들이 개별 활동지에 경제(복지 증진 vs 세금 인하) 및 사회(개인 자유 vs 국가 질서) 관점의 질문에 답하며 자신의 이념 성향을 확인합니다.
- 성향이 유사한 학생들끼리 4~5인 1조를 이루어 정당을 만들고 **정당 이름, 컬러, 슬로건, 핵심 공약**을 정합니다.
- 교사는 웹앱의 **[정당 등록]** 탭에서 조별로 결정한 성향 좌표(-100 ~ 100)와 이름을 등록합니다.
  - *팁: 수업 진행을 미리 테스트해보고자 할 경우, `🚀 빠른 시작 (샘플 4개)` 버튼을 누르면 이념 성향이 서로 대칭되는 대표적인 4개 정당이 즉시 등록됩니다.*

### [2차시] 딜레마 정책 수립과 1차 여론조사
- 교사는 세종국의 유권자 100명 지형을 공개합니다.
- 학생들은 각 유권자 집단(20대 진보 대학생, 60대 보수 은퇴자, 40대 자영업자 중도층 등)의 위치를 보며 어느 표밭을 공략할지 모둠 토론을 진행합니다.
- 각 조는 **100억 원의 가상 예산**을 5대 분야에 배분하여 제출하고, 교사는 이를 웹앱에 입력합니다.
  - *주의: 5개 분야 예산 합계가 정확히 100억 원이어야 입력이 완료됩니다.*
- 교사가 `📊 1차 여론조사 시뮬레이션 실행`을 누르면 카운트다운 오버레이와 함께 개표 방송이 연출되며, 정당 깃발들이 정책 좌표 방향으로 일제히 이동한 뒤 유권자 점들이 지지 정당의 색으로 물듭니다.

### [3차시] 정책 수정, 단일화 협상, 최종 개표
- 1차 여론조사 결과 지지율이 낮게 나온 정당들은 선거 전략을 긴급 수정합니다.
  - **선택 1 (중도 확장)**: 중도층 표를 가져오기 위해 이념 중심인 0점에 가깝게 예산을 재분배합니다.
  - **선택 2 (지지층 결집)**: 확실한 표를 확보하기 위해 기존의 이념 방향(예: 안보 또는 복지 전념)에 돈을 쏟아붓습니다.
  - **선택 3 (단일화/합당)**: 지지성향이 유사한 조끼리 단일화하여 신당을 출범시킵니다.
- 교사는 각 정당의 수정된 예산안 또는 합당 정보를 입력합니다. (수정 전 좌표에서 수정 후 좌표로 향하는 점선 화살표가 지도 상에 투영됩니다.)
- `🗳️ 최종 개표 시작` 버튼을 누르면 집계 방송과 함께 100석의 가상 의회 의석을 나누어 갖는 대망의 최종 승리 정당(1위)이 가려집니다.

## 5. 교사용 발문 힌트 및 수업 토론 주제

| 단계 | 교사 추천 발문 예시 |
| :--- | :--- |
| **정당 등록** | "우리 당이 지향하는 가장 핵심적인 가치는 무엇인가요? 그것을 경제/사회 축으로 표현하면 어디인가요?" |
| **유권자 공개** | "우리 나라의 유권자들은 어느 영역에 가장 조밀하게 모여 있나요? 왜 그런 차이가 발생할까요?" |
| **1차 여론조사** | "중도(0,0)에 예산을 균형 있게 배분한 조와 특정 한 곳에 올인한 조의 지지율이 어떻게 다르게 나왔나요?" |
| **정책 수정** | "신념을 지키기 위해 원래 자리를 유지할 것인가요, 아니면 표를 더 많이 얻기 위해 중도로 이동할 것인가요?" |
| **단일화/합당** | "두 정당의 합당 이후, 기존에 두 정당을 지지하던 유권자들이 정말 그대로 남아있나요? 왜 떠났을까요?" |
| **결과 분석** | "중도를 노린 정당이 승리했다면, 그렇다면 다당제 사회에서 모든 정당이 다 똑같은 중도 공약만 낸다면 어떻게 될까요?" |

## 6. 개발자 가이드 (코드 구조)
본 프로젝트는 외부 서버나 DB 연결이 필요 없는 **클라이언트 사이드 단독 정적 웹앱(Static Web App)**입니다.

- **index.html**: 웹 시뮬레이터의 뼈대 레이아웃. 스테이지별 화면 전환 마크업, 설정 모달, 교사 안내 영역이 HTML5 시맨틱 태그로 정의되어 있습니다.
- **style.css**: 선거 상황실의 긴장감을 연출하기 위한 어두운 테마 스타일 시트. 유권자 파티클의 지지색 글로우 효과, 1위 정당 왕관 펄스 효과, 경로 점선 이동 애니메이션 및 개표 방송 반응형 슬라이드 연출을 제어합니다.
- **app.js**: 시뮬레이션 상태 제어(Stage 1~8) 및 되돌리기(History) 로직, Web Audio API를 활용한 내장 사운드 신디사이저, 이념-예산 환산 벡터 수학 공식 및 유클리드 지지 매칭 엔진을 담고 있습니다.

### 실행 방법
1. 이 저장소의 모든 파일(`index.html`, `style.css`, `app.js`)을 동일한 폴더에 내려받습니다.
2. `index.html` 파일을 더블클릭하여 최신 브라우저(Chrome, Edge, Safari 등)에서 실행하면 즉시 수업에 사용하실 수 있습니다.
